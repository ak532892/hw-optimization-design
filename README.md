## Final Project ##

Solving the Bounded Knapsack Problem with Genetic Algorithm
1. Run main.m

Capacity of the knapsack:20  
Bounded constraint: for all item ≤3  
Generation:50, Population:50  
Crossover rate:0.5, Mutation rate:0.1
